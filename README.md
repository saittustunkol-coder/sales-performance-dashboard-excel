# Sales Performance Dashboard

## Project Overview
This project analyzes sales data using Microsoft Excel to identify revenue trends, top customers, and product performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Dashboard Reporting

## Key Analysis
- Monthly revenue trend
- Top 10 customers
- Top 10 products
- Revenue by country
- Payment status analysis

## Outcome
The project demonstrates data cleaning, analysis, and dashboard creation skills.
